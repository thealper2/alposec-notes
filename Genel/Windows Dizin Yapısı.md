Windows işletim sistemi, dosya sistemi için hiyerarşik bir dizin yapısı kullanır. Bu yapı, tüm dosyaların ve dizinlerin bir ana dizinde başlayarak düzenlendiği ağaç benzeri bir yapıdır. Windows'un dizin yapısı, ağaç yapısı içerisinde kategorize edilen farklı bir sistem ve kullanıcı dosyalarını düzenlemek için kullanılır. Windows işletim sistemi, genellikle bir "C:" sürücüsü olarak adlandırılan ana sürücü üzerinde temel dizin yapısı barındırır.

| Dizin | Açıklama |
| - | - |
| C: | Windows işletim sisteminin yüklü olduğu ana sürücüdür ve tüm diğer dizinler bu sürücü altında düzenlenir. |
| C:\Program Files | 64-bit uygulamaların varsayılan olarak yüklendiği dizindir. |
| C:\Program Files (x86) | 32-bit uygulamaların varsayılan olarak yüklendiği dizindir. |
| C:\Windows | Windows işletim sistemi dosyalarının ve sistem klasörlerinin bulunduğu dizindir. |
| C:\Users | Kullanıcı profillerinin bulunduğu dizindir. Her kullanıcı için ayrı bir alt dizin oluşturur. |
| C:\Documents and Settings | Eski Windows sürümlerinde kullanıcı profillerinin bulunduğu dizindir. Windows 10 ve sonraki sürümlerde kullanılmaz. |
| C:\ProgramData | Tüm kullanıcıların erişebileceği ortak uygulama verilerinin bulunduğu dizindir. |
| C:\Temp | Geçici dosyaların saklandığı dizindir. |
| C:\Windows\System32 | Sistem kütüphanelerinin (DLL) ve sistem komutlarının (EXE) bulunduğu dizindir. |
| C:\Windows\System32\config | Windows kayıt defteri verilerinin bulunduğu dizindir. |
| C:\Windows\Temp | Geçici dosyaların saklandığı bir diğer geçici dizindir. |
| C:\Program Files\Common Files | Birden fazla uygulama tarafından ortak kullanılan dosyaların bulunduğu dizindir. |
